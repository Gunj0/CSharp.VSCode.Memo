# CSharp on VSCode Memo

## dotnet 確認

- `dotnet --version`
  - .NET のインストールバージョン確認
- `dotnet new list`
  - インストールされているテンプレートの確認

## テンプレート作成

- `dotnet new gitignore`
  - .gitignore 作成
- `dotnet new console`
  - コンソールアプリ作成

## デバッグ

- `dotnet new ~` で作成されたフォルダを開くと sln ファイルが作成される
- これを選択して「実行とデバッグ」を押すとデバッグ起動する

- 基本
  - C# (Microsoft)
  - C# Dev Kit (Microsoft)
  - IntelliCode for C# Dev Kit (Microsoft)
